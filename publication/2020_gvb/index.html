Optimal polymer slugs injection profiles

Fedor Bakharev, Aleksandr Enin, Konstantin Kalinin, Yulia Petrova, Nikita Rastegaev, Sergey Tikhomirov
December 2020
PDF Cite Project Project arXiv

The graded viscosity banks technology (tapering) for polymer flooding is studied for several different models of mixing zones behavior. Depending on the viscosity function the limiting polymer injection profile is rigorously derived in case of transverse flow equilibrium assumption, for the Koval model and for the Todd-Longstaff model. The possible gain of the polymer in comparison to finite number of slugs is numerically estimated.
